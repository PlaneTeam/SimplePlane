## 精灵编辑器使用向导

New Sprite  **新建一个精灵**

输入精灵的宽度(int)和高度(int)

输入生成文件的名字

然后进入了编辑精灵的界面



Open Sprite **打开精灵**

输入当前目录下要打开的精灵文件的名字

就可以打开了



Modify Sprite   **编辑精灵**

给出每一种颜色的代码和示例

输入x,y坐标(修改精灵的位置)

以及这个位置的内容bitmap(char)

还有这个位置的前景色(0-f)[偷偷告诉你们，输入超过f的字母程序就崩掉了]

然后就可以看到上面的图案发生了改变



Save Sprite **保存精灵**

退出前请先保存
